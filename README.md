## Window server 2022 installation on Azure cloud

## 1. Sign in to Azure Portal:
    Go to https://portal.azure.com" and sign in with your Azure account credentials.

## 2. Create a Virtual Machine

    Click on "Create a resource" in the upper left corner.
    Now select "Windows Server 2019 Datacenter" and Click "Create" to begin configuring the virtual machine
